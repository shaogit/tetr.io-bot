# 背景素材说明

本目录包含插件使用的背景素材。

## 目录结构

```
backgrounds/
├── base/         # 基础纹理（可平铺）
├── gradients/    # 预生成的渐变背景
├── effects/      # 光效素材
└── patterns/     # 装饰图案
```

## 素材来源

### 推荐的免费素材网站

1. **Unsplash** (https://unsplash.com)

   - 搜索关键词: "dark background", "digital technology"

3. **Subtle Patterns** (https://www.toptal.com/designers/subtlepatterns/)
   - 推荐图案: Carbon Fiber, Hexagons, Circuit Board
   - 许可: CC BY-SA 3.0

4. **Pixabay** (https://pixabay.com)
   - 搜索关键词: "light effect", "glow background"

5. **Hero Patterns** (https://heropatterns.com)
   - SVG 背景图案，可自定义颜色
   - 免费使用

## 所需素材清单

### base/ 目录
- `carbon_fiber.png` (512x512) - 碳纤维纹理
- `hexagon_grid.png` (512x512) - 六角网格
- `tech_lines.png` (512x512) - 科技线条
- `circuit_pattern.png` (512x512) - 电路板图案
- `noise_texture.png` (512x512) - 噪点纹理

### gradients/ 目录
程序自动生成，无需手动添加

### effects/ 目录
- `radial_glow.png` (1920x1080) - 径向光晕
- `edge_light.png` (1920x1080) - 边缘光
- `sparkle.png` (512x512) - 星光闪烁
- `lens_flare.png` (1920x1080) - 镜头光晕

### patterns/ 目录
- `tetromino_watermark.png` (800x800) - 方块水印
- `grid_overlay.png` (512x512) - 网格叠加层
- `corner_decorations.png` (256x256) - 角落装饰

## 使用说明

1. 从上述网站下载合适的素材
2. 使用图像编辑软件（如 Photoshop、GIMP）调整尺寸和颜色
3. 将素材保存到对应目录
4. 重启插件以应用新素材

## 注意事项

- 所有素材必须遵守相应的许可证
- 建议使用深色背景素材，以匹配 TETR.IO 风格
- 素材文件大小建议不超过 1MB
- 支持 PNG、JPG、WebP 格式

## 程序化生成

如果没有素材文件，插件会使用程序化生成的背景：
- 线性/径向渐变
- 几何图案（六角形、方块网格等）
- 纯色背景
