# 字体说明

本目录应包含插件使用的字体文件。

## 所需字体

### 中文字体
- `NotoSansSC-Regular.ttf` - 思源黑体 常规
- `NotoSansSC-Bold.ttf` - 思源黑体 粗体

### 英文/数字字体（可选）
- `RobotoMono-Regular.ttf` - 等宽字体（用于数字显示）
- `Inter-Bold.ttf` - 标题字体

## 下载地址

### Noto Sans SC (推荐)
- 官方网站: https://fonts.google.com/noto/specimen/Noto+Sans+SC
- 下载: https://github.com/googlefonts/noto-cjk/releases
- 许可: SIL Open Font License 1.1

### Roboto Mono
- 官方网站: https://fonts.google.com/specimen/Roboto+Mono
- 许可: Apache License 2.0

### Inter
- 官方网站: https://rsms.me/inter/
- 许可: SIL Open Font License 1.1

## 安装步骤

1. 从上述网站下载字体文件
2. 将 `.ttf` 文件复制到本目录
3. 确保文件名与上述列表一致
4. 重启插件以应用新字体

## 注意事项

- 所有字体文件必须是 TrueType (.ttf) 格式
- 确保字体许可证允许商业使用和分发
- 字体文件大小可能较大，建议只保留所需的字重

## 默认字体

如果没有提供自定义字体，插件将使用 PIL 的默认字体。建议安装自定义字体以获得更好的视觉效果。

## 字体使用场景

- **常规字体**: 正文、标签、说明文字
- **粗体字体**: 标题、用户名、重要数据
- **等宽字体**: 数值、时间、代码
